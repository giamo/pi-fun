# pi-fun

A web app to run on your Raspberry Pi (or any another machine) to transform it into a music/media centre (and more!)

Features
--------
* (In progress) Play music on the server from the Internet or locale storage - connect your speakers to the Raspberry Pi and control what to play from a clean web interface
* (TBD) Voice activation - give commands to your Pi with your voice
* (TBD) Misc features I haven't really thought about yet

How to start
------------

`sbt run`

App will be up at `<hostname>:9000`
